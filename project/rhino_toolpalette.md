# Hub Toolpalette — Rhino Model Catalogue

*Created June 15, 2026. Companion to the Rhino file `toolpalette.3dm` (units: cm).*

This document lists everything in the Rhino toolpalette model 1:1. The model is the
3D **kit-of-parts** for the hub system: every hub element from `hub_typologies.md`
built as a reusable **block**, plus three **sample hub scenes** (S / M / L) assembled
from those blocks in a simple street context.

It is **diagrammatic massing** — each element is built from a few simple volumes at
real scale, enough to read what it is and how it sits in the hub. It is *not* detailed
design; the form of each piece (especially the canopy and the L-hub building) is
resolved later in the typology drawings and the VW case study.

**How it is built / rebuilt:** the whole model is generated by the script
`rhino/build_toolpalette.py`, run inside Rhino through the bridge. Re-running it clears
and rebuilds everything, so the model and the script never drift apart. To edit a piece,
change its builder function in the script and re-run.

## Layer organisation — by material

Elements are coloured by their dominant **material**, not by element type, so the file
reads as a calm, uncrowded material palette. Sub-parts of each block sit on the right
material layer, so inserting a block lands its pieces on the correct layers.

| Layer | Holds | Colour |
|-------|-------|--------|
| `01_Pavement` | Reddish stone field, paving, road surface, markings base | terracotta |
| `02_Concrete` | Kerbs, plinths, foundations, walls, the L-hub structure | warm grey |
| `03_Metal` | Frames, posts, racks, canopy structure, charging posts | blue-grey |
| `04_Wood` | Bench tops, counters, planters, decking, cabinets | brown |
| `05_Glass` | Screens, glazing, shelter panels, vehicle bodies | pale cyan |
| `06_Plastic` | Surface markings, dock units, signage, locker doors | muted yellow |
| `07_Greenery` | Trees, planting, bioswale, living walls, wildflower | green |
| `08_Lighting` | Ground-embedded and ambient glow elements | warm yellow |
| `09_Context` | Carriageway, surrounding buildings (scene context only) | light grey |
| `10_Labels` | Catalogue text + scene titles | dark grey |

Text uses a dedicated annotation style **`TP_Text`** (scale 1, height 50 cm) so labels
read at a sensible size — the file's stock "Centimeters Architectural" style has a 30×
scale that blows text up.

## Block naming

`EL_<category>_<name>` — e.g. `EL_mob_ebikeDock`, `EL_shel_canopy`, `EL_env_treeGrate`.
Each block carries its description, category, materials, tier and zone as **user text**
on its catalogue instance (visible in Rhino's Object Properties → Attribute User Text).

Categories: `mob` mobility · `way` wayfinding/access · `shel` shelter/comfort ·
`env` environment · `soc` social/cultural · `lit` lighting · `str` street · `opt` optional.

## Sample hub scenes

Three typology scenes are assembled from the blocks in the SCENES area of the file
(to the right of / above the catalogue grid). Each sits on its tier-sized reddish stone
field with a simple street context (carriageway + generic building edge on `09_Context`).
Elements are placed by the three-zone logic: Zone 1 vehicle edge → Zone 2 threshold
(canopy) → Zone 3 dwelling edge. Each scene is a named Rhino **group** (`SCENE_S/M/L`).

- **`SCENE_S` — The Moment** (~13 × 18 m). Compact last-mile hub: micro-pod zone,
  e-bike dock, charging, canopy beacon, identity marker, bench, tree, community board,
  art marker, ground lighting, and a hub-integrated crossing over the carriageway.
- **`SCENE_M` — The Choice Point** (~22 × 30 m). Multimodal transfer with buildings on
  both sides: AV drop-off, shared EV bay, cargo dock, info terminal, larger canopy,
  group seating, drinking water, repair, planting/bioswale, kiosk, flexible surface,
  raised carriageway + crossing.
- **`SCENE_L` — The Neighbourhood Anchor** (~40 × 52 m). Repurposed multi-storey car
  park (concrete mass with a glazed ground-floor reveal and green living walls), a
  generous open social field, AV staging and EV fleet charging at the perimeter, plus
  pop-up market, game table, group seating, kiosk, memory marker, info terminal and a
  row of trees.

## What this is for / next steps

This kit is the basis for the typology drawing sheets (plan / section / axonometric per
tier), the detailed VW factory-gate case study, and the Summaery visuals. The blocks are
reusable — drop them onto a real street section to compose any of the 68 hubs.

Open items to refine later: canopy form/character, the L-hub building articulation, and
adapting `SCENE_L` to the actual VW-gate site geometry.
