Team workspace · how it works

The Workspace

A project that remembers itself.

One folder + an AI agent = our single source of truth, our research, and our memory for <stadt.hub> — the Post-Car Future of Wolfsburg.

The problem we kept hitting

Knowledge kept evaporating

💬 Chats end

A great session with the AI… then the chat closes and the reasoning is gone.

🗂️ Files scatter

Numbers in one doc, a decision in another, feedback in a third. Nobody's sure what's current.

🔁 We restart cold

Every session (and every teammate) re-explains the project from zero.

We didn't need more files. We needed a system that holds state.

The idea

The folder is the system

You work in one folder

Everything about the project — concept, evidence, decisions, history — lives in a single repository.

Claude maintains it

The agent reads the rules in the folder, does the work, and writes the result back. It keeps the system tidy for you.

You don't manage the system. You just work — the agent does the bookkeeping.
